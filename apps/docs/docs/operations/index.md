# Runbooks & Operations

JACK ships with an operations toolkit for releasing, monitoring, and coordinating work across teams and agents. Use this section to manage delivery, automation, and release workflows.

- **Spec System**: The Kiro-style spec workflow for feature delivery.
- **Release Flow**: Testnet and mainnet deployment procedures.
- **GitHub Tracker**: Terminal-based mission control for issues and tasks.
