# JACK Video Recorder Placeholder
Place your VRM avatar model here as `jack-avatar.vrm`

## Getting a VRM Model

1. **VRoid Studio** (free): Create a custom avatar at https://vroid.com/en/studio
2. **Ready Player Me**: Generate an avatar at https://readyplayerme.com
3. **Mixamo**: Use Adobe Mixamo for rigged characters (convert to VRM)

Export as `.vrm` format and place in this folder as `jack-avatar.vrm`.
