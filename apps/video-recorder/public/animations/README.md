# JACK Animation Placeholder
Place Mixamo FBX animations here.

## Recommended Animations from Mixamo

1. Go to https://www.mixamo.com
2. Download these animations in FBX format:
   - `Talking.fbx` — Main talking gesture
   - `Idle.fbx` — Standing idle
   - `Waving.fbx` — Greeting wave
   - `Pointing.fbx` — Pointing gesture
   - `Sad.fbx` — Subtle body sway

Export as **FBX Binary (.fbx)** without skin, and place in this folder.
